What is sharesdb？
  sharesdb is an database platform designed  for the Internet of Things (IoT), Finance，Artificial Intelligence . Provide the time-series database, as well provides hybrid transaction/analytical processing、message queuing and other computing functionalities to reduce the complexity and cost of development and operation.  
  sharesdb 是 专为物联网、金融、人工智能等设计的物联网数据库平台。提供完整的时序数据库功能，同时还提供HTAP(Hybrid Transactional/Analytical Processing)、SQL、数据订阅、函数及方程式计算等功能，最大程度减少研发、运维、云部署的复杂度、为企业及客户带来价值。
